# 🇮🇹 Dimenticato - 意大利语背单词应用

一个纯静态的意大利语词汇学习网页应用，支持本地离线使用。

## ✨ 功能特点

### 📚 系统词汇库
- **28,787** 个意大利语单词及英语翻译
- 按使用频率排序（从最常用到最少用）
- 四个难度级别：
  - 🌱 初级：1,000 个高频词
  - 🌿 中级：3,000 个常用词
  - 🌳 高级：5,000 个词汇
  - 🌲 全部：28,787 个完整词库

### 📝 自定义单词本（新功能！）
- **导入自己的单词本** - 支持 JSON 和 TXT 两种格式
- **独立进度跟踪** - 每个单词本的学习进度独立保存
- **添加个人笔记** - 为每个单词添加你自己的学习笔记
- **随时管理** - 导入、删除、切换单词本

#### 📄 TXT 格式 - 超简单的记笔记方式！

不需要处理复杂的 JSON 语法，用记事本就能创建单词本！

**基本规则：**
1. 每个单词用**空行**分隔
2. 每个单词包含 **3-4 行**：
   - 第 1 行：意大利语
   - 第 2 行：英语
   - 第 3 行：中文
   - 第 4 行（可选）：笔记

**示例：**

```txt
ciao
hello, hi, bye
你好，再见
非正式场合使用，既可以表示问候也可以表示告别

grazie
thank you
谢谢
grazie mille = 非常感谢

acqua
water
水

amore
love
爱
意大利文化中非常重要的词汇
```

**优势：**
- ✏️ 用任何文本编辑器都能编辑（记事本、VSCode、Sublime等）
- 🚀 快速记录，不用考虑语法
- 💾 纯文本，方便备份和版本控制
- 📝 想加笔记就加，不加也行

**使用方法：**
1. 创建 `.txt` 文件，按格式添加单词
2. 在应用中点击"导入单词本"
3. 选择你的 TXT 文件
4. 开始学习！

详细说明请查看 [TXT_FORMAT_GUIDE.md](TXT_FORMAT_GUIDE.md)

### 🎯 学习模式

#### ✏️ 选择题模式
- 显示意大利语单词，从4个选项中选择正确的英语翻译
- 智能干扰项生成，提高学习效果
- 即时反馈，显示正确答案

#### ⌨️ 拼写模式
- 显示英语翻译，输入意大利语单词
- 自动检查拼写正确性
- 忽略大小写和重音符号差异

#### 📖 浏览模式
- 查看完整单词列表
- 支持关键字搜索（意大利语/英语）
- 过滤器：全部/已掌握/未掌握
- 显示词频排名

### 💾 进度追踪
- 自动保存学习进度到浏览器本地存储
- 记录已掌握的单词
- 统计学习数据：
  - 总学习单词数
  - 已掌握单词数
  - 学习进度百分比
  - 练习次数和正确率

### 🎨 界面设计
- 简洁现代的用户界面
- 响应式设计，支持手机/平板/电脑
- 深色/浅色主题切换
- 完全中文界面

## 🚀 使用方法

### 超简单！直接双击使用 🎉

1. **下载或克隆本项目到本地**
2. **双击打开 `index.html` 文件**
3. **开始学习！**

就是这么简单！无需安装任何软件，无需运行服务器，无需配置。
词汇数据已经内嵌在 `vocabulary.js` 文件中，可以完全离线使用。

### 备用方法：使用本地服务器

如果遇到任何问题，也可以使用本地服务器：

```bash
# 方法1：Python（推荐）
cd /path/to/Dimenticato
python3 -m http.server 8080
# 然后在浏览器打开 http://localhost:8080

# 方法2：Node.js
npm install -g http-server
http-server -p 8080
# 然后在浏览器打开 http://localhost:8080
```

### 学习流程

#### 使用系统词汇库：
1. **选择难度级别** - 初级/中级/高级/全部
2. **选择学习模式** - 选择题/拼写/浏览模式
3. **开始学习** - 进度自动保存

#### 使用自定义单词本：
1. **创建单词本** - 用记事本创建 `.txt` 文件，按格式添加单词
2. **导入单词本** - 点击"导入单词本"按钮，选择你的文件
3. **选择单词本** - 点击单词本卡片激活
4. **选择学习模式** - 选择题/拼写/浏览模式
5. **开始学习** - 每个单词本的进度独立保存

**提示：** 顶部始终显示当前词汇量、已掌握数量和学习进度

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + 原生 JavaScript
- **数据格式**: JSON
- **存储**: LocalStorage
- **运行平台**: 现代浏览器（Chrome, Firefox, Safari, Edge）

## 📊 数据来源

- 词典数据：意大利语-英语双语词典（28,788 词条）
- 词频数据：基于 ItWaC 语料库的意大利语词频统计（50,000 词）
- 数据处理：自动合并词频和翻译，按使用频率排序

## 🔧 开发者信息

### 数据处理
如果需要重新生成词汇数据：

```bash
node process_data.js
```

这将：
1. 读取 `ita-eng/output.json` 词典文件
2. 读取 `it_50k.txt` 词频文件
3. 合并并清理数据
4. 生成 `data/vocabulary.json` 和 `data/stats.json`

### 数据统计
- 总词汇量：28,787 词
- 有词频数据：11,885 词
- 无词频数据：16,902 词
