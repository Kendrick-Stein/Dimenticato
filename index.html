使用说明

Dimenticato - 背单词

一个纯静态的多语言词汇学习网页应用，当前支持 Italian、German、English 三个语言入口，支持本地离线使用。

功能特点

多语言入口

  • Italian：完整词汇、语法、动词变位、动词搭配、社区与统计体验
  • German：系统词汇练习、德语语法书、基础 Progress、Settings & Data
  • English：系统词汇练习、英语语法书、基础 Progress、Settings & Data
  • 社区词库：当前三个语言入口共用同一个社区词库池

系统词汇库

  • Italian：28,787 个意大利语单词及英语翻译
  • German：已接入德语系统词汇数据，可进行选择题 / 拼写 / 浏览练习
  • English：已接入英语系统词汇数据，可进行选择题 / 拼写 / 浏览练习
  • 系统词汇默认按词频或数据顺序组织，适合从高频内容逐步扩展

自定义单词本

  • 导入自己的单词本 - 支持 JSON 和 TXT 两种格式
  • 独立进度跟踪 - 每个单词本的学习进度独立保存
  • 添加个人笔记 - 为每个单词添加你自己的学习笔记
  • 随时管理 - 导入、删除、切换单词本

TXT 格式 - 超简单的记笔记方式

不需要处理复杂的 JSON 语法，用记事本就能创建单词本！

基本规则：

  1. 每个单词用空行分隔
  2. 每个单词包含 3-4 行：
    • 第 1 行：意大利语
    • 第 2 行：英语
    • 第 3 行：中文
    • 第 4 行（可选）：笔记

示例：

ciao
hello, hi, bye
你好，再见
非正式场合使用，既可以表示问候也可以表示告别

grazie
thank you
谢谢
grazie mille = 非常感谢

acqua
water
水

amore
love
爱
意大利文化中非常重要的词汇

优势：

  • 用任何文本编辑器都能编辑
  • 快速记录，不用考虑语法
  • 纯文本，方便备份和版本控制
  • 想加笔记就加，不加也行

使用方法：

  1. 创建 .txt 文件，按格式添加单词
  2. 在应用中点击"导入单词本"
  3. 选择你的 TXT 文件
  4. 开始学习！

学习模式

  • 选择题模式 - 显示意大利语单词，从4个选项中选择正确的英语翻译
    • 中文提示默认隐藏，需要时点击“显示提示”按钮查看
    • 避免直接给出答案，提高学习效果
  • 拼写模式 - 显示英语翻译，输入意大利语单词
    • 中文提示默认显示，帮助理解单词含义
  • 浏览模式 - 查看完整单词列表，支持搜索和过滤

使用方法

使用系统词汇库：

  1. 先选择语言 - 在左侧 sidebar 中切换 Italian / German / English
  2. 选择难度级别 - 初级/中级/高级/全部
  3. 选择学习模式 - 选择题/拼写/浏览模式
  4. 开始学习 - 进度自动保存

使用自定义单词本：

  1. 先选择语言 - 词本入口会跟随当前语言模块展示
  2. 创建单词本 - 用记事本创建 .txt 文件，按格式添加单词
  3. 导入单词本 - 点击"导入单词本"按钮，选择你的文件
  4. 选择单词本 - 点击单词本卡片激活
  5. 选择学习模式 - 选择题/拼写/浏览模式
  6. 开始学习 - 每个单词本的进度独立保存

提示：顶部始终显示当前词汇量、已掌握数量和学习进度

其他功能

导出/导入学习记录

  • 导出数据 - 将所有学习进度、统计数据、自定义单词本导出为 JSON 文件
    • 包含系统词汇和自定义单词本的学习进度
    • 包含每日统计和练习记录
    • 文件名自动包含导出日期时间
  • 导入数据 - 从之前导出的 JSON 文件恢复学习记录
    • 覆盖模式：清空现有数据，用导入数据完全替换
    • 合并模式：保留现有数据，将导入数据合并进来
    • 适用于跨设备同步学习进度

提示：建议定期导出数据备份；德语和英语虽然使用独立进度 key，但仍走统一导入导出流程。

浏览模式高级功能

  • 添加到单词本 - 在浏览模式中，点击单词旁的"➕"按钮可将单词添加到自定义单词本
    • 可以选择添加到哪个单词本
    • 方便从系统词库中挑选单词创建个性化学习集
  • 搜索过滤 - 支持按意大利语或英语搜索
  • 状态筛选 - 可切换显示全部/已掌握/未掌握单词
  • 查看统计 - 查看学习数据和正确率
  • 切换主题 - 在浅色和深色主题之间切换
  • 重置进度 - 清除所有学习记录（需确认）
