# 📝 TXT 格式单词本使用指南

## 格式说明

TXT 格式让你可以用任何文本编辑器轻松创建和编辑单词本，无需处理复杂的 JSON 语法。

## 基本规则

1. **每个单词用空行分隔**
2. **每个单词包含 3-4 行**：
   - 第 1 行：意大利语单词
   - 第 2 行：英语翻译
   - 第 3 行：中文翻译
   - 第 4 行（可选）：笔记/备注

## 示例格式

### 不带笔记的单词（3行）

```
acqua
water
水

```

### 带笔记的单词（4行）

```
ciao
hello, hi, bye
你好，再见
非正式场合使用，既可以表示问候也可以表示告别

```

## 完整示例

```txt
casa
house
房子
意大利语中最常见的单词之一

libro
book
书
来自拉丁语 liber

acqua
water
水

amore
love
爱
意大利文化中非常重要的词汇

famiglia
family
家庭
意大利人很重视家庭观念
```

## 注意事项

✅ **正确做法**：
- 每个单词之间用一个空行分隔
- 确保每个单词至少有 3 行（意大利语、英语、中文）
- 笔记放在第 4 行（可选）

❌ **常见错误**：
- 忘记在单词之间添加空行
- 某个单词少于 3 行
- 意大利语或英语留空

## 使用方法

1. 用任何文本编辑器创建 `.txt` 文件
2. 按照上述格式添加单词
3. 在应用中点击"导入单词本"按钮
4. 选择你的 `.txt` 文件
5. 应用会自动解析并导入单词本

## 文件命名

文件名将作为单词本的名称显示，例如：
- `意大利语日常用语.txt` → 单词本名称：意大利语日常用语
- `A1词汇.txt` → 单词本名称：A1词汇

## 优势

- 📝 **易于编辑**：用记事本、VSCode 等任何编辑器
- 🚀 **快速记录**：不需要考虑 JSON 语法
- 💾 **便于备份**：纯文本格式，方便版本控制
- 🔄 **灵活性**：随时添加或修改笔记

## 模板下载

在应用中可以下载模板文件：
- `custom_wordbook_template.txt` - TXT 格式模板
- `custom_wordbook_template.json` - JSON 格式模板（功能更强大）
