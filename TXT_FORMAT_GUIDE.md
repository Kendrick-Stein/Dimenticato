# 📝 TXT 格式单词本使用指南

## 格式说明

TXT 格式让你可以用任何文本编辑器轻松创建和编辑单词本，无需处理复杂的 JSON 语法。

**✨ 新功能：智能导入**
- 支持仅输入意大利语单词，系统自动从 28,787 个词库中查找翻译
- 自动跳过重复单词
- 未找到的单词放在最前面，方便手动补充翻译

## 基本规则

1. **每个单词用空行分隔**
2. **每个单词支持 1-4 行灵活格式**：
   - **简化格式（1行）**：仅意大利语 → 自动查找翻译
   - **基础格式（2行）**：意大利语 + 英语翻译
   - **完整格式（3行）**：意大利语 + 英语 + 中文
   - **增强格式（4行）**：意大利语 + 英语 + 中文 + 笔记

## 格式示例

### 🚀 简化格式（推荐！）- 仅意大利语

系统会自动在 28,787 个词库中查找翻译：

```
acqua

casa

libro

ciao

```

如果找到匹配，自动填入英语和中文翻译。
如果未找到，会将单词放在最前面，提示你手动补充。

### 📚 完整格式 - 自定义翻译

适合词库中没有的单词，或你想自定义翻译的情况：

```
casa
house
房子
意大利语中最常见的单词之一

libro
book
书
来自拉丁语 liber

acqua
water
水

```

### 🔀 混合格式

你可以混合使用两种格式：

```
ciao

casa
house
房子
意大利语中最常见的单词之一

acqua

amore
love
爱

```

系统会智能处理：
- 对于仅有意大利语的单词（`ciao`, `acqua`），自动查找翻译
- 对于已提供翻译的单词，使用你的自定义翻译

## 导入处理逻辑

### 🔍 自动查找
- **仅意大利语**：在 28,787 个词库中搜索
- **找到匹配**：自动填充英语和中文翻译
- **未找到**：保留意大利语，英语和中文留空

### 🔄 重复检测
- 如果导入到同名单词本，系统会询问是否合并
- 合并时自动跳过已存在的单词（忽略大小写）
- 显示跳过的重复单词数量

### 📌 智能排序
- **需要手动编辑的单词**（翻译缺失）→ 放在最前面 ⭐
- **完整的单词**（有英语和中文）→ 放在后面

这样打开单词本时，首先看到的就是需要你补充翻译的单词！

## 使用建议

### 💡 快速导入（推荐）

如果单词在系统词库中，只输入意大利语即可：

```txt
ciao

casa

libro

acqua

amore
```

### 📝 自定义翻译

对于特殊用途或词库外的单词，提供完整信息：

```txt
Napoli
Naples
那不勒斯
意大利南部城市

lasciare
to let, to leave
让，离开
常用动词
```

### 🔄 批量更新

可以多次导入同名单词本文件，系统会：
- 自动跳过已有单词
- 只导入新单词
- 显示去重统计

## 使用方法

1. 用任何文本编辑器创建 `.txt` 文件
2. 按照上述格式添加单词（支持简化或完整格式）
3. 在应用中点击"导入单词本"按钮
4. 选择你的 `.txt` 文件
5. 查看导入统计，了解匹配情况
6. 对于需要手动编辑的单词，点击⚙️进入编辑器补充

## 文件命名

文件名将作为单词本的名称显示，例如：
- `意大利语日常用语.txt` → 单词本名称：意大利语日常用语
- `A1词汇.txt` → 单词本名称：A1词汇

## 注意事项

✅ **正确做法**：
- 每个单词之间用一个空行分隔
- 至少提供意大利语单词（第1行必填）
- 如果提供翻译，按顺序：意大利语 → 英语 → 中文 → 笔记
- 可以混合使用简化格式和完整格式

❌ **常见错误**：
- 忘记在单词之间添加空行
- 意大利语单词为空
- 格式顺序错误（如：中文在英语之前）
